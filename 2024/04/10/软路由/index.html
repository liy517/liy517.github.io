PVE虚拟主机 ALL In One

发表于2024-04-10更新于2024-04-10

在软路由安装PVE虚拟机，并在虚拟机中安装WIN10、ROS、homeassistant、openwrt等

一、安装WIN10

1、下载WIN的IOS镜像文件、虚拟光驱virtio-win的IOS镜像文件；

2、将以上两个镜像文件上传到PVE储存

3、创建虚拟机，创建过程中的CD选择WIN10镜像文件，

BIOS中，如果时WIN10选择BIOS，如果是WIN11选择WEFI，并且要勾上TPM，硬盘选scsi性能会更好一点，CPU的类别选择host

创建好后，新建CD/DVD设备，上传virtio，然后再控制台开机。

4、如果识别不到硬盘，点击左下角加载驱动程序，浏览，选virtio的光盘，下拉找到VIOSCSI文件夹打开，选择W10下的amd64文件夹确定即可。

5、安装完成后的网络问题解决：

打开“我的电脑”进入virtio驱动光盘，找到virtio的安装程序，双击，一路下一步

二、安装ROS

1、下载地址：MikroTik Routers and Wireless - Software

2、下载CHR版本（Cloud Hosted Router）

3、将下载的IMG文件上传到PVE的IOS镜像，上传完成后，复制保存路径，在shell里输入

qm importdisk 104 /var/lib/vz/template/iso/chr-7.13.3.img local

/var/lib/vz/template/iso/jump78uefi.img

/var/lib/vz/template/iso/openwrt-23.05.img

qm importdisk 103 haos_ova-11.5.qcow2 local

qm importdisk 101 VM110.qcow2 local

然后回到PVE后台，硬件里有一个未使用的硬盘，双击 ——添加——选项——勾选刚才的硬盘

在控制台启动

七、PVE分区调整

在PVE的shell内或SSH终端内运行以下命令移除local-lvm对应的逻辑卷.

lvremove pve/data

接下来运行以下命令对local对应的逻辑卷pve/root进行在线扩容.

lvextend -l +100%FREE -r pve/root

然后在数据中心-存储里删除local-lvm

如果添加新的磁盘，在磁盘里选择该磁盘，擦除，使用GPT初始化，目录里创建

八、无网络安装pvestools

项目地址（GitHub）：https://hub.fastgit.org/ivanhao/pvetools

命令行安装：

用root账号运行

删除企业源：rm /etc/apt/sources.list.d/pve-enterprise.list

终端中按行执行以下内容：

export LC_ALL=en_US.UTF-8

apt update && apt -y install git && git clone https://hub.fastgit.org/ivanhao/pvetools.git

cd pvetools

./pvetools.sh

无脑安装：

echo “nameserver 223.5.5.5” >> /etc/resolv.conf && rm /etc/apt/sources.list.d/pve-enterprise.list && export LC_ALL=en_US.UTF-8 && apt update && apt -y install git && git clone https://hub.fastgit.org/ivanhao/pvetools.git && cd pvetools && ./pvetools.sh

以上无法安装可使用 MobaXterm 连接 PVE 后 将下载并解压好的 pvetools 导入 pve 根目录内 （pvetools 文件 在项目地址里下载）

运行工具：./pvetools.sh

如果提示没有权限，输入：chmod +x ./*.sh

查看可直通列表：lspci -nn

全盘映射指令

查看硬盘列表 ：ls /dev/disk/by-id

硬盘映射：qm set 100 -sata1 /dev/disk/by-id/ （100为虚拟机序号，-sata 支持 0-5 ）
